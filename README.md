# Secrets Project

## Description
A Client-side web app that uses the [Secret API](https://secrets-api.appbrewery.com) in order to reveal people's secret anonymously

## Objective
- To learn how to work with Public APIs.
- Understanding API docs.
- How to use axios to make request to APIs.
- How to create an Express server.

## Tools
- EJS
- Axios
- Secret API
- JavaScript
- CSS
